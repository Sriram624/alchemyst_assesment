# Strap in — the next launch window is open!

## Pick your seat on the ship
For the crew to break orbit, here's the hats we need to fill!

**BASE EXPECTATION**: Exceptionally high agency, adaptability and learnability.
**Salary is no bar**.

## Cloud Cartographer
**Commonly known as**: DevOps Intern / Infra Engineer

[**Assignment Link**](./devops/devops-internship-assignment.md)

### Superpowers we want
- Carve out private subnets and wire VMs together like it's second nature.
- Stand up reproducible infra — clicks in the console don't count, IaC does.
- Keep workers off the public internet while still letting the front door breathe.
- Sense the difference between "it works on my VM" and "it works on a fresh account at 2am".
- **Skills (Brownie points)**: AWS/GCP, Terraform/Pulumi, VPC + subnet design, systemd/containers, Python/TypeScript, basic RPC plumbing.

## Realtime Whisperer
**Commonly known as**: B2B Custom Agent Engineer

[**Assignment Link**](./b2b-custom-agent-assignment.md)

### Superpowers we want
- Make streaming feel alive — tokens, interruptions, idle nudges, the whole feel of a phone call rendered in text.
- Tame WebSockets so reconnects and partial state don't ruin the illusion.
- Track conversation context tight enough to know *where* an interruption landed, not just *that* one happened.
- Ship a UI that customers can't put down — chat tabs, dark mode, typing indicators, all of it.
- **Skills (Brownie points)**: WebSockets, streaming LLM APIs (OpenAI/Gemini/Anthropic), React/Next.js, state machines, latency-aware UX.

## Agentic Architect
**Commonly known as**: Fullstack AI Engineer

[**Assignment Link**](./fullstack-ai-assignment.md)

### Superpowers we want
- Build agentic systems that *don't* hallucinate — guardrails, structured outputs, and "not mentioned" beats a confident lie.
- Treat context as a first-class citizen: extracted data, conversation history, and intent all live in structured memory.
- Know when to reach for a tool vs. when to let the LLM cook.
- Wrap the whole thing in a clean, modular fullstack app with a UX that makes the intelligence obvious.
- **Skills (Brownie points)**: TypeScript/Python, OpenAI/Claude APIs, agent frameworks, PDF/text parsing, schema-driven outputs, fullstack delivery (bonus: WebRTC for the optional voice mode).
